

## Clone this project

```bash
> git clone https://github.com/jhonataT/WebScraping
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd WebScraping
> npm i
```
### Usage
```bash
> npm dev run
```
